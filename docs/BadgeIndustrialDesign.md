# 安全工牌工业设计需求文档

## 1. 产品定位

安全工牌是 SafePost 的一线入口，佩戴在装卸工腰部或胸前，用于身份识别、作业监护、主动报警和状态提醒。

它不是装饰性工牌，而是高强度物流现场使用的安全设备。

## 2. 佩戴方式

优先形态：腰夹式。

设计要求：

- 可夹在腰带、裤腰、反光背心或工作服边缘。
- 跑动、弯腰、搬货时不易脱落。
- 不应影响装卸动作。
- 支持挂绳孔作为备用佩戴方式。

## 3. 外观要求

- 造型简洁，现场一眼识别为安全设备。
- 正面保留醒目的 SOS 按键。
- 状态灯必须在 3 米外可见。
- 工牌编号和二维码/NFC 标识清晰。
- 颜色建议：高可见橙/安全黄 + 深灰防污外壳。

## 4. 人机交互

### 4.1 状态灯

| 状态 | 展示 |
| --- | --- |
| 允许作业 | 绿灯常亮或间歇闪烁 |
| 待检测 | 黄灯常亮 |
| 禁止作业 | 红灯常亮 |
| 紧急报警 | 红灯爆闪 |
| 低电量 | 黄灯慢闪 |

### 4.2 SOS 按键

- 按键必须大，戴手套也能按。
- 建议长按 2 秒触发，避免误触。
- 触发后震动、蜂鸣、红灯爆闪。
- 按键周围应有凹陷或护圈，降低搬货误触。

### 4.3 确认按键

- 用于无响应检测时确认“我安全”。
- 可与 SOS 分离，避免误操作。
- 确认按键尺寸小于 SOS，但应易于盲按。

## 5. 结构要求

- 外壳抗摔，目标 1.5 米跌落不损坏核心功能。
- 防尘防汗，目标 IP54 起步。
- 边角圆润，避免刮伤皮肤或衣物。
- 腰夹支持可更换设计。
- USB-C 充电口需有防尘塞。

## 6. 电子硬件

建议配置：

- MCU：ESP32-C3。
- 通信：BLE、Wi-Fi，后续可扩展 ESP-NOW。
- 传感：三轴加速度计。
- 输出：RGB LED 或三颗独立高亮 LED、蜂鸣器、震动马达。
- 输入：SOS 按键、确认按键。
- 电池：满足至少 1 个班次，建议目标 12 小时以上。
- 标识：二维码、NFC 或印刷唯一 ID。

## 7. 可靠性

- 支持低电量提醒。
- 支持离线缓存最近关键事件。
- 网络断开后应持续本地提醒。
- 报警状态需要人工或云端明确解除。

## 8. 量产关注点

- 腰夹耐久性。
- 按键寿命。
- 灯光可视角度。
- 蜂鸣器在嘈杂场景中的可听性。
- 震动在厚工作服下的可感知性。
- 电池安全和充电管理。

## 9. 设计验收

- 一线工人 5 分钟内理解红黄绿状态。
- 戴手套可完成 SOS 和确认操作。
- 3 米外可辨认状态灯。
- 1.5 米跌落后仍可报警。
- 连续工作时长满足 1 个班次。
