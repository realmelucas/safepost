# 硬件规格

## 1. ESP32-C3 工牌

建议配置：

- MCU：ESP32-C3。
- 通信：Wi-Fi、BLE，现场可扩展 ESP-NOW。
- 显示：红/黄/绿三色 LED。
- 输入：SOS 按键、确认按键。
- 输出：蜂鸣器或震动马达。
- 电源：锂电池，USB-C 充电。
- 标识：二维码/NFC/印刷唯一编号。
- 传感：加速度计可选，用于无响应和跌倒辅助判断。

状态定义：

| 状态 | LED | 说明 |
| --- | --- | --- |
| PENDING | 黄灯 | 待检测或检测未完成 |
| ALLOWED | 绿灯 | 允许上岗 |
| FORBIDDEN | 红灯 | 禁止上岗 |
| ALERT | 红灯闪烁 | SOS 或云端报警 |

## 2. ESP32-S3 基站

建议配置：

- MCU：ESP32-S3。
- 网络：Wi-Fi，量产可选以太网。
- 现场通信：BLE/ESP-NOW/Wi-Fi MQTT。
- 输出：蜂鸣器、状态灯、可选小屏。
- 安装：月台、车厢入口、检测点。

职责：

- 接收或转发工牌心跳。
- 向工牌广播通知。
- 将检测结果、在线状态、报警消息转发到云端。
- 在网络异常时缓存关键消息。

## 3. 检测设备接口

血压计和酒精检测设备可以通过以下方式接入：

- 串口接入基站。
- BLE 接入基站。
- HTTP API 直接上传。
- 人工录入作为 MVP 备用方案。

## 4. 安全建议

- 每个设备预置 deviceId 和 deviceSecret。
- MQTT 使用 TLS 与账号密码或证书。
- 设备只订阅自身相关 Topic。
- 工牌本地保存最近一次状态，但以云端状态为准。
